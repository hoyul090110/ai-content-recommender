# AI 콘텐츠 추천 시스템 데모

과제 발표용 정적 웹사이트입니다. OpenAI API, 서버, 데이터베이스 없이 브라우저에서 바로 실행됩니다.

## 주요 기능

- 키워드 기반 콘텐츠 추천
- 좋아요, 북마크, 공유, 관심없음 활동 기록
- 긍정 활동 기반 개인화 추천
- 관심없음 표시 주제 추천 점수 감소
- 브라우저 localStorage를 활용한 활동 저장
- 반응형 카드형 UI

## 배포 방법

GitHub 저장소 최상위에 아래 파일을 업로드합니다.

```text
index.html
styles.css
app.js
package.json
README.md
```

Vercel 또는 Netlify에서 정적 사이트로 배포하면 됩니다.

## 프로젝트 설명

사용자가 입력한 키워드와 콘텐츠 태그의 유사도를 계산하여 관련 콘텐츠를 추천합니다. 사용자가 좋아요, 북마크, 공유한 콘텐츠의 키워드는 긍정 관심사로 반영되고, 관심없음을 누른 콘텐츠의 키워드는 추천 점수를 낮추는 방식으로 개인화 추천을 구현했습니다.
