# AI Content Recommender

사용자가 입력한 키워드를 바탕으로 AI가 실제 웹 콘텐츠를 검색해 추천하고, 좋아요·북마크·공유·관심없음 활동을 반영해 개인화 추천을 제공하는 Vercel용 웹 프로젝트입니다.

## 핵심 기능

- 키워드 입력 기반 실제 웹 검색 추천
- 기사, 영상, 이미지 자료 추천
- 콘텐츠 카드 썸네일 표시
- 좋아요 / 북마크 / 공유 / 관심없음
- 긍정 표시한 여러 키워드를 융합한 맞춤 추천
- 관심없음 키워드 추천 비중 감소
- OpenAI API 연결 실패 시 기본 추천 모드 제공

## 파일 구조

```text
ai-content-recommender/
├── index.html
├── styles.css
├── app.js
├── package.json
├── README.md
└── api/
    └── recommend.js
```

## Vercel 환경변수

Vercel 프로젝트 설정에서 아래 환경변수를 추가합니다.

```text
OPENAI_API_KEY=새로 발급받은 OpenAI API 키
OPENAI_MODEL=gpt-4.1-mini
```

환경변수 저장 후 반드시 Redeploy 해야 합니다.

## 배포 방법

1. GitHub 레포지토리의 기존 파일을 삭제합니다.
2. 이 ZIP 안의 파일을 저장소 최상위에 업로드합니다.
3. `api/recommend.js` 구조가 정확한지 확인합니다.
4. Vercel에서 GitHub 저장소를 연결하거나 Redeploy 합니다.
5. 사이트에 접속해 키워드를 입력합니다.

## 과제 설명 예시

이 웹사이트는 사용자가 입력한 키워드를 기반으로 AI가 실제 인터넷 콘텐츠를 검색하여 기사, 영상, 이미지 자료를 추천합니다. 또한 사용자가 좋아요, 북마크, 공유한 콘텐츠의 키워드를 종합해 관심사를 분석하고, 관심없음으로 표시한 주제는 추천 비중을 낮춰 개인화된 새로운 콘텐츠를 제공합니다.
