추천 알고리즘 설명

1. 사용자 관심사 점수

선택한 관심 카테고리는 기본 점수를 얻습니다.

2. 활동 기반 점수

조회 1점, 좋아요 3점, 북마크 4점, 공유 5점으로 반영합니다.

3. 콘텐츠 유사도

태그와 카테고리가 사용자 활동과 비슷한 콘텐츠를 추천합니다.

4. 콜드 스타트

활동이 없으면 기본 인기 콘텐츠와 선택한 관심사를 함께 사용합니다.
